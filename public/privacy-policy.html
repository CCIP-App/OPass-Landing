隱私權政策版本： 2020-08-14 2.0

我們 OPass 尊重並承諾保護您的個人資料以及隱私，當您使用 OPass 的服務即代表同意 OPass 的隱私政策，
若您對此份隱私政策有任何疑問請至 https://github.com/CCIP-App/issues 開立 ISSUE 詢問，或者是來信至 dev _at_ opass.app

1. 我們所蒐集的資料及蒐集的方式

當您使用 OPass 服務，我們可能會搜集您的
  1. 開啟相機或相簿
  2. 票券本身代碼（Token）
  如果您有報名現場活動的話主辦單位可能會取得您留下的聯絡資訊或是姓名，OPass 只負責轉傳資訊，不會儲存相關資訊。
  3. 應用程式操作行為

2. 我們如何使用您的資訊

我們使用您的相機或是相簿來進行 QR Code 辨識，圖片結果並不會上傳雲端，您也可以不使用這項服務使用手動方式輸入票券 Token

應用程式操作行為由 Firebase 進行收集，提供開發者了解應用程式使用狀況

3. 我們會與誰分享這些資訊

伺服器由各活動主辦單位維護，所在國家請洽尋活動主辦單位，於其中我們使用了 GitHub, Cloudflare, Firebase 以及 OneSignal 的服務，
您的票卷資訊有可能透過這些服務提供者進行傳輸。

GitHub 的隱私權政策（英文）： https://help.github.com/en/articles/github-privacy-statement
Cloudflare 的隱私權政策（英文）： https://www.cloudflare.com/privacypolicy/
Firebase 的隱私權政策： https://firebase.google.com/support/privacy
OneSignal 的隱私權政策： https://onesignal.com/privacy_policy
